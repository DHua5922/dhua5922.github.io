# Dylan Hua Portfolio

Personal portfolio website for Dylan Hua, built as a static GitHub Pages site with HTML, CSS, and vanilla JavaScript.

## Overview

The site includes:

- Responsive About, Projects, Contact, and Footer sections
- Light/dark theme toggle persisted with `localStorage`
- Animated background videos with poster image fallbacks
- Project cards rendered from static data
- Project detail dialog with responsive image sources
- Contact form validation and EmailJS submission

## Tech Stack

- HTML5
- CSS3
- JavaScript
- [Font Awesome](https://fontawesome.com/)
- [EmailJS](https://www.emailjs.com/)

## Running Locally

Because this is a static site, you can open `index.html` directly in a browser.

## Deployment

This repository is configured for GitHub Pages. The custom domain is defined in `CNAME`.

Push changes to the `main` branch for the repository to deploy updates.
